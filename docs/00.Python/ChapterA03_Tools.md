## 包的可执行文件导出

pyinstaller

安装

pip3 install pyinstaller

使用：

pyinstaller -F -w -i ico图片路径 py文件路径
选项：
-F  表示生成单个可执行文件；
-i  icofile表示给程序加上图标，图标必须为.ico格式；
-w  表示隐藏程序运行时的命令行窗口（不加-w会有黑色窗口）；
